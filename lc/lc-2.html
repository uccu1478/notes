題意：給兩個ListNode*(singly-linked list)，相加，head為個位數
輸入：兩個ListNode*
輸出：ListNode*
Testcase：
  1. Input: l1 = [2,4,3], l2 = [5,6,4] Output: [7,0,8] Explanation: 342 + 465 = 807.
  2. Input: l1 = [0], l2 = [0] Output: [0]
  3. Input: l1 = [9,9,9,9,9,9,9], l2 = [9,9,9,9] Output: [8,9,9,9,0,0,0,1]
